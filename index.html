Welcome to Broadside

This is the page for Broadside, an Android Mobile Game developed in Eclipse. -Matthew O'Dea

This was developed in Fall 2013.

Design

All graphics were original, and created by Matt O'Dea using Adobe Illustrator.

Coding

Coding was done as a collaboration effort, by a group of 9, all in raw Java. This was especially a challenge without a game engine, done in a semester. The group still managed to hard code path systems, artificial intelligence, and smooth animation and overall gameplay.
